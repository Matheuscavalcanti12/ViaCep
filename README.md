# 📍 API ViaCEP - Projeto Básico

Este é um projeto simples que consome a API pública do **ViaCEP** para buscar informações de endereço a partir de um CEP.

## 🚀 Funcionalidades

* Consulta de endereço pelo CEP
* Retorno de dados como:

  * Logradouro
  * Bairro
  * Cidade
  * Estado




## 📦 API utilizada

* https://viacep.com.br/
